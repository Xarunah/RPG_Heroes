# RPG Heroes
The player can choose between 4 different classes.
* Mage
* Ranger
* Rogue
* Warrior

Each class can wear different amor and weapon types.

#### Armor
* Cloth
* Leather
* Mail
* Plate

#### Weapon
* Staff
* Bow
* Dagger
* Axe

Every character start at lvl 1 and need 1000 XP to level up.
